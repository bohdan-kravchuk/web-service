# Project Installation

### Backend
1. Create a copy of `.env.example` file and name it `.env`.
1. Update values in `.env` file with your local settings, if necessary.
1. Run the following commands:
```
npm install
npm start
```
### Frontend

1. Update values in `.env` file with your local settings, if necessary.

